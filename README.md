## 代码提交规范
#### Commit message 前缀规范提要
| code      | info          
|---------- |-------------- |
| **feat**: msg | 新功能 feature | 
| **fix**: msg | 修复bug | 
| **merge**: msg | merge 信息 | 
| **docs**: msg | 文档修改 | 
| **style**: msg | 格式，不影响代码运行的变动 | 
| **refactor**: msg | 重构 | 
| **test**: msg | 增加测试 | 
| **chore**: msg | 构建过程或辅助工具的变动 | 
| **rm**: msg | 删除文件或代码 |
